# DSA Toolkit (Java)

A **menu-driven console application** in Java that allows users to practice and execute basic Data Structures and Algorithms programs.  
This toolkit is designed to strengthen programming and problem-solving skills for beginners and intermediate learners.

---

## Features
1. **Binary Search** – Search an element in a sorted array efficiently.  
2. **Bubble Sort** – Sort an array using Bubble Sort algorithm.  
3. **Stack (using Array)** – Implement stack operations: Push, Pop, Peek, Display.  
4. **Queue (using Array)** – Implement queue operations: Enqueue, Dequeue, Display.  
5. **String Reverse** – Reverse any string input by the user.  
6. **Palindrome Check** – Check whether a string is a palindrome.

---

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Arrays
- Loops
- Conditional Statements
- Menu-driven Console Interface

---

## How to Run
1. Clone or download the repository.
2. Open in VS Code (or any Java IDE).
3. Compile `DSAToolkit.java`.
4. Run the program and follow the **menu** prompts.
